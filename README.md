# 📊 Sales Forecast Project

A modular Python-based data analysis and machine learning project that analyzes store revenue and predicts future growth.

---

## 🚀 Features

- 📈 Revenue analysis for multiple stores
- 🏆 Current best performing store detection
- 🤖 Machine Learning (Linear Regression) for future prediction
- 🔮 Future best store prediction
- 📊 Visualization with Matplotlib (2 graphs: original vs predicted)
- 🧩 Modular code structure (analysis, ML, visualization)
